# harness_enchanting

happy ghast harnesses now accept these vanilla enchantments via an anvil:
- **depth strider** (flying speed)
- **soul speed** (more flying speed, in the nether)

the flying speed depends on the levels of applied enchantments:
- **depth strider I-III**: +0.25, +0.50, +0.75
- **soul speed I-III**: +0.10, +0.17, +0.25
